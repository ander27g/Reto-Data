Tipos de datos

Un tipo de datos es la propiedad de un valor que determina su dominio (qué valores puede tomar), qué operaciones se le pueden aplicar y cómo es representado internamente por el computador. Todos los valores que aparecen en un programa tienen un tipo. A continuación vamos a revisar los tipos de daros que se usan en html, css y javascript.

Datos Html

En Html se escribe en forma de «etiquetas», rodeadas por corchetes angulares. En HTML también puede describir, hasta un cierto punto, la apariencia de un documento, y puede incluir o hacer referencia a un tipo de programa llamado script, el cual puede afectar el comportamiento de navegadores web y otros procesadores de HTML. HTML consta de varios elementos y sus atributos, en el siguiente link se pueden explorar todos los elementos y algunos de sus principales atributos: https://developer.mozilla.org/es/docs/Web/HTML/Element

El elemento HTML <input> se usa para crear controles interactivos para formularios basados en la web con el fin de recibir datos del usuario. Hay disponible una amplia variedad de tipos de datos de entrada que continuación vamos a describir:

Datos CSS

Los tipos de datos básicos de CSS definen los valores típicos (incluidas las palabras clave y las unidades) aceptados por las propiedades y funciones de CSS. Son un tipo especial de valor de componente.En sintaxis formal, los tipos de datos se denotan con una palabra clave colocada entre los signos de desigualdad;
